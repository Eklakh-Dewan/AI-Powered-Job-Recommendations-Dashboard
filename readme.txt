# AI-Powered Job Recommendations Dashboard

## Project Overview
This is an intelligent job recommendation system that uses AI and natural language processing to match your resume with relevant job opportunities.

## Features
- Resume upload and text extraction
- AI-powered semantic job matching
- Real-time job recommendations
- Country-based job filtering
- Dynamic job trend visualization

## Prerequisites
- Python 3.8+
- Virtual Environment (recommended)

## Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/job-recommendations-dashboard.git
cd job-recommendations-dashboard
```

2. Create a virtual environment
```bash
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Run the Streamlit app
```bash
streamlit run app.py
```

## Technologies Used
- Streamlit
- Sentence Transformers
- Remotive Jobs API
- PyPDF2
- Pandas
